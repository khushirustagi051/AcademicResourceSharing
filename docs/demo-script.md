# Demo Script (CampusVault)

## Demo Goal
Show that CampusVault is a structured academic resource hub where students can:
- upload resources
- discover using filters
- download
- rate and review
- bookmark
- see verified badge
- see contributor leaderboard

---

## Demo Flow (Recommended Order)

### 1) Landing Page (10 seconds)
- Explain the problem: resources scattered across WhatsApp and drive links
- Show the CTA: Browse / Upload / Leaderboard

---

### 2) Login (15 seconds)
- Login with demo account:
  - Email: demo@campus.com
  - Password: demo123

Explain: only logged-in users can upload/review/bookmark.

---

### 3) Home Feed (30 seconds)
- Show resources feed
- Show filters:
  - Subject
  - Semester
  - Type
  - Year (QP)
- Show sorting:
  - Highest rated
  - Newest
  - Year

---

### 4) Resource Details (30 seconds)
- Open a resource
- Show:
  - uploader name
  - rating summary
  - reviews list
  - verified badge (if present)

---

### 5) Download (10 seconds)
- Click download button
- Explain: download count increases

---

### 6) Rate + Review (20 seconds)
- Add rating + review
- Explain: one review per user per resource

---

### 7) Bookmark (10 seconds)
- Bookmark the resource
- Open bookmarks page
- Show it appears there

---

### 8) Verified Badge (15 seconds)
- Show a verified resource
- Explain the rule:
  - avg rating >= 4.2 AND reviews >= 5

---

### 9) Leaderboard (20 seconds)
- Open leaderboard page
- Explain points:
  - Upload = +10
  - Verified bonus = +20

---

## Closing (10 seconds)
CampusVault turns informal resource sharing into a structured campus learning ecosystem.
